Model evolution v3.1 — patch for RefDsl.xtext
============================================

File: part1/tool2-xtext/pt.isep.enorm.refdsl/src/pt/isep/enorm/refdsl/RefDsl.xtext

Find the RefModel rule:

  RefModel returns RefModel:
      'RefModel' name=EString
      ('version' version=EString)?
      '{'

Change to:

  RefModel returns RefModel:
      'RefModel' name=EString
      ('version' version=EString)?
      ('description' description=EString)?
      '{'

Then in Eclipse:
  1. Right-click GenerateRefDsl.mwe2 → Run As → MWE2 Workflow
  2. Project → Clean → pt.isep.enorm.refdsl
  3. Re-run Eclipse Application
  4. Open examples/amazon-v3.0.refdsl (valid without description)
  5. Open examples/amazon-v3.1.refdsl (valid with description)

Optional: extend buildTextProjection in RefDslGenerator.xtend to print description in .txt projection.
