USER TYPES:
-----------
- User (kind: GENERIC)
- Creator (kind: CREATOR)
- Moderator (kind: MODERATOR)

RESOURCE TYPES:
---------------
Resource Channel
  - id: TEXT (required)
  - name: TEXT (required)
  - description: TEXT
  - creationDate: DATE (required)

Resource Content
  - id: TEXT (required)
  - creationDate: DATE (required)
  - status: TEXT (required)

Resource Video
  Supports multimedia content
  - title: TEXT (required)
  - description: TEXT (required)
  - uploadDate: DATE (required)
  - duration: NUMBER (required)
  - commentStatus: TEXT (required)

Resource Comment
  - text: TEXT (required)

Resource LikeHistory
  - id: TEXT (required)
  - action: TEXT (required)
  - timestamp: DATETIME (required)

Resource VideoModerationCheck
  - id: TEXT (required)
  - type: TEXT (required)
  - result: TEXT (required)
  - timestamp: DATE (required)

Resource CommentModerationCheck
  - id: TEXT (required)
  - type: TEXT (required)
  - result: TEXT (required)
  - timestamp: DATE (required)

Resource CommentSettingsChange
  - id: TEXT (required)
  - newStatus: TEXT (required)
  - timestamp: DATE (required)

Resource ChannelPermissionPolicy
  - id: TEXT (required)
  - action: TEXT (required)
  - allowedFor: TEXT (required)

  - id: TEXT (required)

FEEDBACK SYSTEM:
----------------
Feedback type Like
  Kind: VOTE
  Subject scope: RESOURCE_ONLY

Feedback type Subscription
  Kind: SUBSCRIPTION
  Subject scope: RESOURCE_ONLY

Feedback type Report
  Kind: REPORT
  Subject scope: RESOURCE_ONLY


Feedback definition LikeOnContent
  Type: Like
  Author: User

Feedback definition SubscriptionToChannel
  Type: Subscription
  Author: User

Feedback definition ReportOnContent
  Type: Report
  Author: User


AUTHORIZATION RULES:
-------------------
- Creator can CREATE Video
- User can COMMENT Comment
- User can VOTE Content
- User can REPORT Content
- User can SUBSCRIBE Channel
- Moderator can MODERATE Content

MODERATION POLICIES:
-------------------
Moderation policy VideoModerationPolicy
  Mode: HYBRID
  Decision: FLAGGED
  Trigger: ON_REPORT_SUBMITTED

Moderation policy CommentModerationPolicy
  Mode: HYBRID
  Decision: HIDDEN
  Trigger: ON_FEEDBACK_CREATE

Validation rule CommentPolicyValidation
