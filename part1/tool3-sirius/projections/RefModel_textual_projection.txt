==============================================================
ENORM Model: DSL
Version: 1.0.0
==============================================================

USER TYPES:
-----------
- Buyer (kind: BUYER)
- Seller (kind: SELLER)
- Creator (kind: CREATOR)
- Moderator (kind: MODERATOR)
- Admin (kind: GENERIC)

RESOURCE TYPES:
---------------
Resource Product
  Supports multimedia content
  - title: TEXT (required)
  - description: TEXT (required)
  - price: NUMBER (required)
  - category: TEXT (required)
  - images: IMAGE
  - videoUrl: VIDEO

Resource Review
  Supports multimedia content
  - title: TEXT (required)
  - content: TEXT (required)
  - rating: NUMBER (required)
  - purchaseDate: DATE (required)
  - helpful: NUMBER

Resource Comment
  - text: TEXT (required)
  - timestamp: DATETIME (required)

Resource Shop
  Supports multimedia content
  - name: TEXT (required)
  - description: TEXT
  - rating: NUMBER

Resource User
  - name: TEXT (required)


CONTEXT TYPES:
--------------
Context Global (kind: GLOBAL)
  Contains:
    - Product
    - Review
    - Comment
    - Shop
    - User

Context Marketplace (kind: COMMUNITY)
  Contains:
    - Product
    - Shop
    - User

Context ProductReviews (kind: CHANNEL)
  Contains:
    - Review
    - Comment


FEEDBACK SYSTEM:
----------------
Feedback type ProductReview
  Kind: REVIEW
  Subject scope: RESOURCE_ONLY
  Has rating capability

Feedback type ReviewComment
  Kind: COMMENT
  Subject scope: RESOURCE_OR_FEEDBACK
  Supports recursive feedback

Feedback type Reaction
  Kind: REACTION
  Subject scope: RESOURCE_OR_FEEDBACK

Feedback type Vote
  Kind: VOTE
  Subject scope: RESOURCE_OR_FEEDBACK

Feedback type Complaint
  Kind: REPORT
  Subject scope: RESOURCE_OR_FEEDBACK


Feedback definition ProductRating
  Type: ProductReview
  Author: Buyer
  Requires verified context

Feedback definition ReviewComment
  Type: ReviewComment
  Author: Buyer

Feedback definition SellerReport
  Type: Complaint
  Author: Buyer
  Requires verified context


AUTHORIZATION RULES:
-------------------
- Buyer can READ Product
- Seller can CREATE Product
- Seller can UPDATE Product
- Moderator can DELETE Comment
- Buyer can RATE Review
- Buyer can COMMENT Comment
- Moderator can MODERATE Review

MODERATION POLICIES:
-------------------
Moderation policy SpamDetectionPolicy
  Mode: AUTOMATIC
  Decision: FLAGGED
  Trigger: ON_FEEDBACK_CREATE

Moderation policy OffensiveContentPolicy
  Mode: HYBRID
  Decision: HIDDEN
  Trigger: ON_FEEDBACK_CREATE

Moderation policy FakeReviewPolicy
  Mode: MANUAL
  Decision: REMOVED
  Trigger: ON_REPORT_SUBMITTED


VALIDATION RULES:
-----------------
Validation rule ProductPriceValidation
  Kind: AUTOMATIC
  Severity: ERROR
  Expression: self.price > 0

Validation rule ReviewRatingRange
  Kind: AUTOMATIC
  Severity: WARNING
  Expression: self.rating >= 1 and self.rating <= 5

Validation rule ReviewMinLength
  Kind: AUTOMATIC
  Severity: ERROR
  Expression: self.content.size() >= 10

Validation rule CommentNotEmpty
  Kind: AUTOMATIC
  Severity: ERROR
  Expression: self.text.size() > 0

Validation rule ShopNameRequired
  Kind: MANUAL
  Severity: ERROR


AUTOMATION RULES:
-----------------
Automation rule LowRatingNotification
  Trigger: ON_FEEDBACK_CREATE

Automation rule HighRatingBadge
  Trigger: ON_FEEDBACK_CREATE

Automation rule ReviewLengthEscalation
  Trigger: ON_FEEDBACK_CREATE

Automation rule CommentQualityCheck
  Trigger: ON_FEEDBACK_CREATE

Automation rule ShopProfileCheck
  Trigger: ON_RESOURCE_CREATE
