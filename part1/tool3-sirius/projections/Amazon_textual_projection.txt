USER TYPES:
-----------
- User (kind: GENERIC)
- Buyer (kind: BUYER)
- Seller (kind: SELLER)
- Moderator (kind: MODERATOR)

RESOURCE TYPES:
---------------
Resource Product
  - id: TEXT (required)
  - name: TEXT (required)
  - description: TEXT (required)

Resource Order
  - id: TEXT (required)
  - date: DATE (required)

Resource OrderItem
  - quantity: NUMBER (required)

Resource MediaAttachment
  Supports multimedia content
  - id: TEXT (required)
  - type: TEXT (required)
  - url: URL (required)

Resource VerificationProcess
  - id: TEXT (required)
  - type: TEXT (required)
  - status: TEXT (required)
  - timestamp: DATETIME (required)

Resource HelpfulVote
  - id: TEXT (required)
  - timestamp: DATETIME (required)

Resource ReviewModerationCheck
  - id: TEXT (required)
  - reason: TEXT (required)
  - result: TEXT (required)
  - timestamp: DATETIME (required)


CONTEXT TYPES:
--------------
Context Catalog (kind: CATALOG)
  Contains:
    - Product
    - Order
    - OrderItem

Context FeedbackSpace (kind: CHANNEL)
  Contains:
    - MediaAttachment
    - VerificationProcess
    - HelpfulVote
    - ReviewModerationCheck


FEEDBACK SYSTEM:
----------------
Feedback type ProductReview
  Kind: REVIEW
  Subject scope: RESOURCE_ONLY
  Has rating capability

Feedback type HelpfulVote
  Kind: VOTE
  Subject scope: FEEDBACK_ONLY

Feedback type ReviewModerationCheck
  Kind: REPORT
  Subject scope: FEEDBACK_ONLY


Feedback definition ProductReviewOnProduct
  Type: ProductReview
  Author: Buyer
  Requires verified context

Feedback definition HelpfulVoteOnReview
  Type: HelpfulVote
  Author: User

Feedback definition ModerationCheckOnReview
  Type: ReviewModerationCheck
  Author: Moderator


AUTHORIZATION RULES:
-------------------
- Buyer can CREATE Product
- User can VOTE Product
- Moderator can MODERATE Product

MODERATION POLICIES:
-------------------
Moderation policy ReviewModerationPolicy
  Mode: HYBRID
  Decision: HIDDEN
  Trigger: ON_REPORT_SUBMITTED


VALIDATION RULES:
-----------------
Validation rule ReviewVerifiedPurchaseValidation
  Kind: AUTOMATIC
  Severity: ERROR
  Expression: review_author_has_verified_purchase

Validation rule HelpfulVoteConsistencyValidation
  Kind: AUTOMATIC
  Severity: WARNING
  Expression: helpful_vote_not_duplicate_and_target_is_review


AUTOMATION RULES:
-----------------
Automation rule ReviewSubmissionValidation
  Trigger: ON_FEEDBACK_CREATE

Automation rule HelpfulVoteValidation
  Trigger: ON_FEEDBACK_CREATE
